Invoke the `setup-agentic-workflow` skill to inspect this workspace and either:

- Set up the workflow for the first time (fresh installs).
- Sync the local environment to an existing team install (for new collaborators joining a project).
- Complete a partial install left behind by a previous run.
- Update the team's shared workflow config (with branch + PR for shared file changes).
- Upgrade an existing install to the current plugin version by walking through migration files.

Follow the skill's full instructions — inspect first, ask before writing, default to "do as little as possible."
